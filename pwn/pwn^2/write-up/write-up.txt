В данном задании подразумевалась обратная разработка не только основной программы, но и баш-скрипта, который запускался программой.

При запуске программы она требует ключ, предварительно прочитав файл secret.

Сравнив введёный пароль с содержимым файла, программа выдаёт файл flag при условии равенства этих строк.

Содержимое файла secret изменяется баш-скриптом, который был обфусцирован.

Его можно деобфусцировать или просто исполнить.

Если исполнить скрипт в обфусцированном виде, то возникнет две ошибки:

bash: line 1: /home/pwn3/secret: No such file or directory
md5sum: /home/pwn3/keyholder: No such file or directory

Можно предположить, что в файл secret кладётся md5 хэш от файла keyholder, который дан игроку.

Если проверить эту гипотезу, то пароль окажется верным.

Postmortem: изначально предполагалась обфускация проще, чтобы деобфускация была легче
